--- layout: post title: 'Awesome Tool of the Day: Burp Suite' date: '2012-05-22T15:34:00.000-07:00' author: Daniel Drummond tags: modified_time: '2012-05-22T15:34:33.317-07:00' thumbnail: http://4.bp.blogspot.com/-FU2g7mdWdQY/T7wSmjMJi8I/AAAAAAAAB0Q/a_RAg3szTLI/s72-c/burp1.png blogger_id: tag:blogger.com,1999:blog-531773883684728825.post-6223134549114421860 blogger_orig_url: http://apporia.blogspot.com/2012/05/awesome-tool-of-day-burp-suite.html --- Burp suite is a collection of tools designed to allow for the testing of web applications.  It is a java tool, and happily runs on both Windows and Linux.  I tested it on Windows 7 and Fedora, and it worked well on both. It comes in two versions: a free version with a limited featureset, and a paid version containing all features.



My particular need for it today was sniffing https traffic to investigate a protocol.  The component which allows this is Proxy.  The way it works is by redirecting all your traffic through a local proxy server set up on port 8080.  For https traffic the proxy provides its own certificates, which means it is able to read the packets.  It then communicates securely with the remote site securely using the real certificate.  This can be used for a multitude of purposes including reverse engineering a protocol, or debugging applications that use https.

There are a ton of other features that this tool offers, I really haven't had a chance to look at much of it.  If the rest of it is as useful and simple to use as Proxy, this is one tool you need to have.
